Целью данного проекта было создание приложения, которое умеет расшифровывать
введенное (заведомо зашифрованное) пользователем сообщение.

В данной версии сообщение может быть расшифровано следующими способами:
1. Шифр Цезаря с неизвестным ключом
2. Шифр Цезаря
3. Шифр Атбаш
4. Метод статистического анализа
5. Шифр Гронсфельда
Для того, чтобы определить способ расшифровки, пользователю необходимо знать
метод, которым сообщение было зашифровано. Это не относится к Методу
статистического анализа. Метод статистического анализа работает корректно
с любым моноалфавитным методом шифрования. Однако, зашифрованный текст
должен быть достаточно объёмным.
На данном этапе приложение работает только с кириллицей.

Приложение написано с помощью технологии WPF и паттерна MVP.
